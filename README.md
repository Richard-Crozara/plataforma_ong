# Instituto Esperança — Sistema de Cadastro de Voluntários

Projeto desenvolvido para a disciplina **Desenvolvimento Front-End Para Web - Turma 001** da **Cruzeiro do Sul**.  
O objetivo é criar um site completo utilizando HTML, CSS e JavaScript, aplicando boas práticas de acessibilidade, semântica e versionamento com Git/GitHub.

## Descrição do Projeto

O site foi criado para representar uma ONG fictícia chamada **Instituto Esperança**, que atua em ações sociais e ambientais.  
O projeto inclui páginas com informações sobre o Instituto, formulário de cadastro de voluntários e uma área de contato.

## Funcionalidades

- Navegação entre páginas  
- Formulário de cadastro de voluntários  
- Layout responsivo  
- Navegação por teclado e suporte para leitores de tela  
- Estrutura semântica e acessível

## Boas Práticas Aplicadas

- Uso de tags semânticas (`header`, `main`, `section`, `footer`)  
- Contraste de cores adequado  
- Foco visível e navegação por teclado  
- Código comentado e organizado  
- Estrutura de arquivos limpa e separada (HTML, CSS e JS)
